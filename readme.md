# Project Title (Update)

Employee Database in SQLite


## Instructions for Build and Use

Steps to build and/or run the software:

1. The SQLite library is already included in Python, so just run the script and you're all set

Instructions for using the software:

1. Run the script
2. Use the menu options to add, update, search, list, or delete employees
